# Инструкция по запуску

## Важно! Приложение сейчас запущено

Перед выполнением команд ниже **остановите текущий процесс приложения** (закройте окно или нажмите Ctrl+C в терминале).

## Шаги для запуска обновленного приложения:

### 1. Остановите приложение
Закройте текущий запущенный процесс clunComp.exe

### 2. Удалите старую базу данных
```bash
cd clunComp
Remove-Item app.db -Force
```

### 3. Удалите старые миграции
```bash
Remove-Item -Recurse -Force Data/Migrations
```

### 4. Создайте новую миграцию
```bash
dotnet ef migrations add InitialCreate
```

### 5. Примените миграцию
```bash
dotnet ef database update
```

### 6. Запустите приложение
```bash
dotnet run
```
