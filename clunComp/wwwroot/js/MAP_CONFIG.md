# Настройка карты

## Как изменить координаты клуба:

1. Откройте файл `site.js`
2. Найдите строки:
   ```javascript
   const clubLat = 55.7558;
   const clubLng = 37.6173;
   ```
3. Замените на координаты вашего клуба

## Как узнать координаты:

### Способ 1: Google Maps
1. Откройте Google Maps
2. Найдите ваш адрес
3. Кликните правой кнопкой на точку
4. Выберите "Что здесь?"
5. Координаты появятся внизу

### Способ 2: Яндекс.Карты
1. Откройте Яндекс.Карты
2. Найдите ваш адрес
3. Кликните правой кнопкой
4. Выберите "Что здесь?"
5. Координаты появятся в левой панели

## Пример координат городов России:
- Москва: 55.7558, 37.6173
- Санкт-Петербург: 59.9343, 30.3351
- Казань: 55.7887, 49.1221
- Екатеринбург: 56.8389, 60.6057
- Новосибирск: 55.0084, 82.9357

## Настройка адреса в popup:
В файле `site.js` найдите строку с popup и измените адрес и телефон на реальные.
