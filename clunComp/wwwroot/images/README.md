# Изображения для зон

## Карточки зон:
1. `standard-zone.jpg` - фото общей зоны с игровыми ПК
2. `premium-zone.jpg` - фото лучших мест с топовым оборудованием

## Слайдер галереи:
Замените `placeholder.svg` на реальные фотографии клуба:
- Можно использовать любые названия (gallery-1.jpg, gallery-2.jpg и т.д.)
- Обновите пути в файле `Pages/Index.cshtml` в секции `.gallery-slider`

Рекомендуемый размер: 
- Карточки зон: 800x600px или больше
- Слайдер: 1200x500px или больше
Формат: JPG или PNG
