# Киберспорт Клуб - Веб-приложение

Полнофункциональное веб-приложение компьютерного клуба на ASP.NET 9 Razor Pages с современным темным дизайном.

## ✨ Возможности

### Для пользователей:
- 🎮 Просмотр доступных мест по зонам (VIP, Стандарт, PS)
- 📅 Бронирование с выбором даты и времени
- 📱 Простая форма (только телефон, дата, время)
- 👤 Регистрация и вход в систему
- 📋 Просмотр своих бронирований
- ❌ Отмена бронирований

## 🚀 Запуск приложения

### Первый запуск:

```bash
cd clunComp

# Создать миграцию
dotnet ef migrations add InitialCreate

# Применить миграцию (создаст базу данных)
dotnet ef database update

# Запустить приложение
dotnet run
```

Приложение будет доступно по адресу: https://localhost:5001

### Если база уже создана:

```bash
cd clunComp
dotnet run
```

## 🔐 Доступ к админ-панели

Админ-панель доступна по адресу: `/Admin/Index`

### Функции админ-панели:

1. **Управление бронированиями**
   - Просмотр всех ожидающих бронирований
   - Подтверждение бронирований
   - Отмена бронирований
   - Завершение бронирований

2. **Управление местами**
   - Включение/отключение доступности мест
   - Просмотр информации о всех местах

3. **Статистика**
   - Количество ожидающих бронирований
   - Количество подтвержденных бронирований
   - Количество доступных мест

### Как попасть в админку:

Просто перейдите по ссылке в меню "Админ" или откройте:
```
https://localhost:5001/Admin/Index
```

**Примечание:** В текущей версии админ-панель доступна всем пользователям. Для продакшена рекомендуется добавить авторизацию с проверкой роли администратора.

### Добавление защиты админ-панели (опционально):

Если нужно ограничить доступ только для администраторов, добавьте в `Admin/Index.cshtml.cs`:

```csharp
[Authorize(Roles = "Admin")]
public class IndexModel : PageModel
{
    // ...
}
```

И создайте роль администратора при первом запуске.
